## フォルダ構造

`hackclubaicli .`コマンドで、指定フォルダを初期化：

```
/
├── .hackclubaicli/
│   ├── settings.json          # プロジェクト固有の設定
│   ├── chats.json         # 会話履歴
```

## スラッシュコマンド (`/`)

### セッション管理系

テキストインプットから`/`で始まらないコマンドを入力すると、通常の Gemini モデルへのプロンプトとして扱われます。

**`/chats`**

- without arg - 現在のプロジェクトで保存された会話を table で表示。
  - `f`で検索ボックスにフォーカス
  - `l`でリストにフォーカス
  - `Enter`で選択した会話を復元
  - `s`でお気に入りマークの切り替え
  - `d` `d` で選択した会話を削除（confirm を挟む）
  - `e`でエクスポート（エクスポートタイプのセレクタを出す）
  - `F2`で選択した会話のタイトルを編集（ダイアログでAI生成/手入力を選択）
- with arg `new` - 新しい会話を開始
- プロンプトにファイル添付がある場合もしくは画像生成タグがついている場合、呼び出すAPIエンドポイントを変更

### 設定・カスタマイズ系

- settings.json ファイルを直接編集する

**`/models`** - 使用可能なモデルをリスト表示

- `f`で検索ボックスにフォーカス
  - クエリ: #[key]:[value] 形式でフィルタリング可能 (例: `#type:chat #favorite:true`)
- `l`でリストにフォーカス
- `Enter`で選択したモデルで会話をするよう設定
- `s`でお気に入りマークの切り替え
- `r`でモデルの役割を設定するダイアログ（セレクタ）を表示（default for chat, titling, default for image）

**`/stats`** - チャットごとの統計情報を table 表示

- トークン使用量、キャッシュされたトークンの節約量、セッション時間など
- ディレクトリ全体の統計情報も表示

**`/help`** - ヘルプ情報を表示

**`/about`** - バージョン情報を表示

**`/quit`** - CLI を終了

## At コマンド (`@`)

ファイルの内容をプロンプトに含めるコマンド

**`@<file name>`** - 指定したファイルまたはディレクトリの内容をプロンプトに注入

- file name 入力時にオートコンプリートが利用可能（レンダリングはバッチのファイル名）
- ディレクトリは除外
- 隠しファイルやバイナリファイルは除外

## Slarah コマンド (`#`)

Slarah コマンドは、特定のタスクを実行するためのプリセットコマンドです。

**`#imageGenerate`** - 画像生成タスクを実行

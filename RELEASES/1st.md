## Running Instructions

### Linux
1. Download the latest release using wget:
   ```bash
   wget https://github.com/chi1180/Hack-Club-AI-CLI/releases/latest/download/cli-tool-linux
   ```
2. Make it executable:
   ```bash
   chmod +x cli-tool-linux
   ```
3. Run the tool:
   ```bash
   ./cli-tool-linux
   ```

### macOS
1. Download the latest release using curl:
   ```bash
   curl -LO https://github.com/chi1180/Hack-Club-AI-CLI/releases/latest/download/cli-tool-macos
   ```
2. Make it executable:
   ```bash
   chmod +x cli-tool-macos
   ```
3. Run the tool:
   ```bash
   ./cli-tool-macos
   ```

### Windows
1. Download the latest release:
   Visit the [GitHub releases page](https://github.com/chi1180/Hack-Club-AI-CLI/releases) and download `cli-tool-windows.exe`.
2. Run the tool by double-clicking the downloaded file or using the command line.

# Hack Club AI CLI

Hack Club AI CLI is a command-line tool designed to help you harness the power of AI. This project aims to support learning and development of AI applications for the Hack Club community.

## Features

- Easy execution and usage of AI models
- Control AI directly from the command line
- Includes example scripts and configuration files for experiments and learning
- Connected with the Hack Club community for support

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/chi1180/Hack-Club-AI-CLI.git
    ```
2. Install dependencies:
    ```bash
    # Example
    pip install -r requirements.txt
    ```

## Usage

Run the CLI tool inside the project directory to perform various AI-related actions. You can check available commands and options with:

```bash
python cli.py --help
```
